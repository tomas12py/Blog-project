## Proyecto para la creación y la búsqueda de viajes
Este proyecto tiene como concepto la creación de viajes para la búsqueda de diferentes lugares del mundo que el usuario quiera visitar. El usuario puede definir sus viajes realizados para dar un feedback de su experiencia a través de la descripción de su viaje y una calificación del mismo. De esta forma, el usuario puede compartir su experiencia y buscar experiencias en distintos lugares del mundo de otros usuarios.

<br>

## Estructuras utilizadas

- Componetización
- Manejo del estado de los datos
- Consumo de una API
- Manejo de rutas
- Estructuras para UX
- DRY

## Como inicializar el proyecto

Instala las dependecias de npm usando este comando:

`npm install`


Corre el proyecto en el entorno de desarrollo con este comando:

`npm run dev`


<br>

## Funcionalidades de la aplicación

### Pagina de inicio

La página de inicio tiene información sobre la utilización y el propósito de la aplicación, mostrando una vista previa de lo que el usuario podrá hacer. La aplicación cuenta con un navbar para la navegación entre rutas y un footer para la información de contacto y redes sociales.​

### Login
La aplicación permite la creación de un usuario, validando que los valores introducidos cumplan con el estándar requerido para cada campo y generando una notificación cuando el usuario se crea correctamente. Además, la aplicación permite realizar un login básico, verificando que el usuario tenga sus credenciales registradas en la API.

### Gestión de los viajes

La aplicación permite una correcta gestión de los viajes, validando sus campos desde su creación y generando una notificación al ser creados. Además, permite modificar los datos deseados, eliminar viajes específicos y visualizar todos los viajes creados.
